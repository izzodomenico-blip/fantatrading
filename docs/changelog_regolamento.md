Ogni modifica al regolamento deve essere documentata qui con:

2. **Parametro** modificato
3. **Valore precedente → Valore nuovo**
4. **Motivazione** (risultato di simulazione o feedback utenti)
5. **Impatto atteso** (calcolato tramite analisi di sensibilità)
